- Edge (145/144/143/142以上)
- Firefox (147/146/145/140以上)
- Chrome (145/144/143/142/139/133/131/125/112/109以上)
- Safari (26.3/26.2以上)
- Opera (125/124以上)
- Node.js (25.1.0/24.11.0/22.21.0/20.19.0以上)
